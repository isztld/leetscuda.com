## Background

CUDA (Compute Unified Device Architecture) is NVIDIA's parallel computing platform that lets you write programs that execute across thousands of GPU cores simultaneously. The fundamental unit of work in CUDA is the **kernel** — a function that runs on the GPU and is executed by many threads at once.

Threads in CUDA are organised into a hierarchy: threads are grouped into **blocks**, and blocks are grouped into a **grid**. Each thread has a unique identity given by `threadIdx` (position within its block) and `blockIdx` (the block's position in the grid). Using these two built-in variables, a thread can compute a global index into the data it is responsible for.

## What to Implement

Implement the `vecAdd` CUDA kernel that adds two floating-point arrays element-wise:

```
c[i] = a[i] + b[i]   for every i in [0, n)
```

Each thread should handle exactly one element. Use the standard 1-D indexing formula:

```cpp
int i = blockIdx.x * blockDim.x + threadIdx.x;
```

Guard against out-of-bounds access with `if (i < n)` — the last block in the grid may have threads beyond the array length.

## Constraints

- `1 ≤ n ≤ 10,000,000`
- Arrays `a` and `b` contain 32-bit floats in the range `[-10^6, 10^6]`
- Use 256 threads per block (the host code already sets this up for you)
- Do not modify the host setup code in `solve()`

## Example

**Input:** `n = 4`, `a = [1.0, 2.0, 3.0, 4.0]`, `b = [10.0, 20.0, 30.0, 40.0]`

**Output:** `c = [11.0, 22.0, 33.0, 44.0]`

Each element `c[i]` is the sum of `a[i]` and `b[i]`. Thread 0 computes `c[0] = 1 + 10 = 11`, thread 1 computes `c[1] = 2 + 20 = 22`, and so on.

// Host wrapper — allocates device memory, copies data, launches kernel, copies result back.
// Do NOT modify anything below this line.
void solve(const float* h_a, const float* h_b, float* h_c, int n) {
    float *d_a, *d_b, *d_c;
    cudaMalloc(&d_a, n * sizeof(float));
    cudaMalloc(&d_b, n * sizeof(float));
    cudaMalloc(&d_c, n * sizeof(float));

    cudaMemcpy(d_a, h_a, n * sizeof(float), cudaMemcpyHostToDevice);
    cudaMemcpy(d_b, h_b, n * sizeof(float), cudaMemcpyHostToDevice);

    const int blockSize = 256;
    const int gridSize  = (n + blockSize - 1) / blockSize;
    vecAdd<<<gridSize, blockSize>>>(d_a, d_b, d_c, n);

    cudaMemcpy(h_c, d_c, n * sizeof(float), cudaMemcpyDeviceToHost);
    cudaFree(d_a);
    cudaFree(d_b);
    cudaFree(d_c);
}

---solution---
__global__ void vecAdd(const float* a, const float* b, float* c, int n) {
    int i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) {
        c[i] = a[i] + b[i];
    }
}
