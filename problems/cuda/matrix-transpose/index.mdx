## Background

One of the most important performance considerations in CUDA is **memory coalescing**. When threads in a warp access global memory, the hardware can merge those accesses into a single transaction only if the addresses are contiguous. A naïve matrix transpose reads rows (coalesced) but writes columns (uncoalesced), cutting throughput by up to 32×.

The standard fix is to use **shared memory** as a staging buffer. Each block loads a tile from the input matrix into shared memory — reading contiguous rows, so accesses are coalesced. It then writes the *transposed* tile to the output matrix by reading columns from shared memory, which are only `__syncthreads()` away and do not touch slow global memory.

## What to Implement

Implement `transpose`, a tiled CUDA kernel that transposes an N×N matrix:

```
out[col][row] = in[row][col]
```

Use a `TILE × TILE` shared memory buffer. Each block handles one tile:
1. Load `in[row][col]` into `smem[ty][tx]` (coalesced read from global memory)
2. `__syncthreads()` to ensure the tile is fully loaded
3. Write `smem[tx][ty]` to `out[blockIdx.x*TILE+tx][blockIdx.y*TILE+ty]` (coalesced write)

To avoid **shared memory bank conflicts** when reading the transposed tile, declare the shared buffer with a padding column: `float smem[TILE][TILE + 1]`.

## Constraints

- `N` is a multiple of `TILE` (you may assume `TILE = 32`)
- `1 ≤ N ≤ 4096`
- The matrix is stored in **row-major** order; element `(r, c)` is at offset `r * N + c`
- Do not modify the host code in `solve()`

## Example

**Input:** 2×2 matrix `[[1, 2], [3, 4]]` (N=2)

**Output:** `[[1, 3], [2, 4]]`

Element `(r, c)` of the output equals element `(c, r)` of the input.

    size_t bytes = (size_t)N * N * sizeof(float);
    cudaMalloc(&d_in,  bytes);
    cudaMalloc(&d_out, bytes);
    cudaMemcpy(d_in, h_in, bytes, cudaMemcpyHostToDevice);

    dim3 grid(N / TILE, N / TILE);
    transpose<<<grid, block>>>(d_in, d_out, N);

    cudaMemcpy(h_out, d_out, bytes, cudaMemcpyDeviceToHost);
    cudaFree(d_in);
    cudaFree(d_out);
}

__global__ void transpose(const float* in, float* out, int N) {
    __shared__ float smem[TILE][TILE + 1];

    int col_in  = blockIdx.x * TILE + threadIdx.x;
    int row_in  = blockIdx.y * TILE + threadIdx.y;

    if (col_in < N && row_in < N)
        smem[threadIdx.y][threadIdx.x] = in[row_in * N + col_in];

    __syncthreads();

    int col_out = blockIdx.y * TILE + threadIdx.x;
    int row_out = blockIdx.x * TILE + threadIdx.y;

    if (col_out < N && row_out < N)
        out[row_out * N + col_out] = smem[threadIdx.x][threadIdx.y];
}
